add_lnpdf(lnlikelihood)[source]¶

Adds the argument to the log probability

A method of multiplying self.PDF by another probability distribution by adding the logarithms. Useful in calculating Bayes rule, with a likelihood when its logarithm is available. Updates both PDF and lnPDF. Normalization is not enforced. See also multiply_pdf().

Parameters

lnlikelihood – An array representing the log of a probability distribution.

pdf_config(paramvals)[source]¶

initializes the probability distribution function

Sets up an N-dimensional numpy array as a discrete representation of a joint probability. Assumes that the probability is zero outside the parameter space defined by paramvals.

Parameters

paramvals (tuple of lists, required) – Each list (or array) in paramvals contains allowed values of a parameter. Individual lists may represent values of a naturally discrete parameter, or they may be discrete approximations of continuous parameters.

set_pdf(flat=False, probvalarrays=None, pdf=None, lnpdf=None)[source]¶

Initializes the probability distribution function

Initializes the distribution using the first non-default argument. Normalization is not enforced. Initializes both PDF and lnPDF.

Parameters

  • flat (bool) – If True, sets PDF values to one and lnPDF values to zero. Defaults to False.

  • probvalarrays (tuple of arrays) – Initializes PDF to the outer product of the input arrays. Dimensions of probvalarrays members must match dimensions of paramvals arrays. Defaults to None.

  • pdf (list or ndarray) – Specifies the values of PDF directly. Defaults to None.

  • lnpdf (list or ndarray) – Specifies the values of lnPDF directly. Defaults to None.
